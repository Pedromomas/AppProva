Sobre o TaskFlow

TaskFlow v1.0

Este aplicativo foi desenvolvido como parte de um projeto de demonstração, utilizando as seguintes tecnologias:

Angular

Framework para construção da interface de usuário.

Ionic Framework

Toolkit de UI para desenvolvimento multiplataforma.

Google Firebase

Backend-as-a-Service para banco de dados (Firestore).

Criado para demonstrar integração de frontend e backend em um aplicativo moderno.
